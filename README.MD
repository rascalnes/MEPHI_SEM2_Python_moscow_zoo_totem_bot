# 🐾 Тотемное животное Московского зоопарка — Telegram бот-викторина

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://python.org)
[![Telegram Bot](https://img.shields.io/badge/Telegram-Bot-blue.svg)](https://telegram.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

Telegram-бот викторина для популяризации программы опеки Московского зоопарка. Бот помогает пользователям найти своё тотемное животное через увлекательную викторину и рассказывает о возможности стать опекуном.

## ✨ Особенности

- 🎮 **Интерактивная викторина** — 6 вопросов с выбором ответа
- 🦁 **5 тотемных животных** — манул, белый медведь, пингвин, фламинго, рысь
- 📸 **Персонализированный результат** — фото животного + описание + интересный факт
- 💚 **Программа опеки** — информация о том, как стать опекуном животного
- 📤 **Поделиться результатом** — готовый текст для соцсетей со ссылкой на бота
- 🔄 **Перезапуск викторины** — можно проходить сколько угодно раз
- ⭐ **Обратная связь** — пользователи могут оставлять отзывы
- 📞 **Контакты зоопарка** — быстрая связь с сотрудниками

## 🛠 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/rascalnes/MEPHI_SEM2_Python_moscow_zoo_totem_bot
cd MEPHI_SEM2_Python_moscow_zoo_totem_bot
```

### 2. Создание и активация виртуального окружения

Windows:
```bash
python -m venv .venv
.venv\Scripts\activate
```

### 3. Установка зависимостей

```bash
python -m venv .venv
.venv\Scripts\activate
```

Linux/Mac:
```bash
python3 -m venv .venv
source .venv/bin/activate
```

```bash

pip install -r requirements.txt
```

### 4. Создание бота в Telegram

* Откройте Telegram и найдите @BotFather
* Отправьте команду /newbot
* Придумайте имя бота (например, Московский зоопарк бот)
* Придумайте username бота (должен заканчиваться на bot, например, moscow_zoo_totem_bot)
* Сохраните полученный токен

### 5. Настройка конфигурации

Создайте файл config.py на основе примера:
# config.py
# Токен бота (получите у @BotFather)
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"

# Контактные данные зоопарка (для примера)
ZOO_CONTACT_EMAIL = "opeka@moscowzoo.ru"
ZOO_CONTACT_PHONE = "+7 (499) 255-00-93"
ZOO_MANAGER_USERNAME = "moscowzoo_manager"
